## Pagination types
For type variations, the following css classes are built:
- `pagination--outline` for standard pagination
- `pagination--outline-negative` for standard pagination on dark background

<Canvas>
  <button type="button" class="pagination pagination--outline">1</button>
</Canvas>

<Canvas style={{backgroundColor: '#2f4356'}}>
  <button type="button" class="pagination pagination--outline-negative">1</button>
</Canvas>

## Sizes
For size variations, the following css classes are built:
- `pagination--sm` for small buttons
- `pagination--base` for standard size
- `pagination--lg` for large buttons

<Canvas>
  <button type="button" class="pagination pagination--outline pagination--sm">1</button>
  <button type="button" class="pagination pagination--outline pagination--base">2</button>
  <button type="button" class="pagination pagination--outline pagination--lg">3</button>
</Canvas>
