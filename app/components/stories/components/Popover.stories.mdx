# Popover

## Example

popovers are used to attach an extra information to a text, a label or a title. If you need interactive element, use the [tagItem](?path=/docs/components-tagitem--example) instead.

## Colors
For color variations, the following css classes are built:
- `popover--info`
- `popover--error`
- `popover--warning`
- `popover--success`

and

- `popover--gray`
- `popover--red`
- `popover--yellow`
- `popover--orange`
- `popover--green`
- `popover--blue`
- `popover--indigo`
- `popover--purple`
- `popover--pink`

## Sizes
For size variations, the following css classes are built:
- `popover--base` default value
- `popover--sm` for small popovers

    <button class="popover popover--base popover--gray">popover--base</button>
