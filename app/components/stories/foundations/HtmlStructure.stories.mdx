# Navigations overview

Let's have a closer look at the main header:

```html
<header id="main-header">
  <div class="top-bar" />
  <div class="top-header" />
  <div class="mainmenu" />
  <div class="mobilemenu" />
</header>
```
The three first blocks contain a navigation in a desktop context:

- `.top-bar` contains the `.top-bar-navigation`
- `.top-header` contains the `.meta-navigation`
- `.mainmenu` contains the `.main-navigation`

In a mobile context though, these three navigations are all regrouped in the `.mobilemenu` container.

Regarding the implementation approach, the content of the navigations may be duplicated in the `.mobilemenu` container. A better approach would be to swap their positions between the mobilemenu and their desktop containers.
