# Fonts

## Headings
For heading variations, the following css classes are built:
- `.h1` for heading 1
- `.h2` for heading 2
- `.h3` for heading 3
- `.h4` for heading 4
- `.h5` for heading 5

  <div>
    <div class="h1 font--bold">Heading 1</div>
    <div class="h2 font--bold">Heading 2</div>
    <div class="h3 font--bold">Heading 3</div>
    <div class="h4 font--bold">Heading 4</div>
    <div class="h5 font--bold">Heading 5</div>
  </div>
</Canvas>

## Text sizes
For size variations, the following css classes are built:
- `text--lg` for lg
- `text--base` for base
- `text--sm` for sm
- `text--xs` for xs

<Canvas>
  <div>
    <div class="text--lg">lg</div>
    <div class="text--base">base</div>
    <div class="text--sm">sm</div>
    <div class="text--xs">xs</div>
  </div>
</Canvas>

## Styles and Weights
For style variations, the following css classes are built:
- `font--regular` for regular
- `font--italic` for italic
- `font--bold` for bold
- `font--bold-italic`  for bold italic

<Canvas>
  <div>
    <div class="font--regular">Regular</div>
    <div class="font--italic">Italic</div>
    <div class="font--bold">Bold</div>
    <div class="font--bold-italic">Bold italic</div>
  </div>
</Canvas>
