# Containers

`.container` classes and its variants give a structured horizontal constraint to the content. Depending of the viewport, it combines maximal width and horizontal paddings to define content's width in a coherent way.

`.container` is frequently a child of a `.section` element.

<Canvas className="storybook-demo-grid">
  <section>
    <div class="container">
      Container with horizontal padding and limited width on large viewport
    </div>
  </section>
</Canvas>

# Container with flexbox

- `container--flex` displays its content horizontally

<Canvas className="storybook-demo-grid">

    <div class="container container--flex">
      <div>Child 1</div>
      <div>Child 2</div>
      <div>Child 3</div>
    </div>
  </section>
</Canvas>

# Container with 12 cols grid

`.container--grid` creates a 12 columns CSS grid. It defines the main layout within a strict but flexible structure. 

## Centered sub-containers

Use the following classes for `container--grid` children:
- `container__full` fullfills the 12 columns
- `container__center--md` fills 10 columns on large viewports 
- `container__center--sm` fills 8 columns on large viewports
- `container__center--xs` fills 6 columns on large viewports 

Additionally, a `gap--responsive` class applies default spacings between container--grids.

<Canvas className="storybook-demo-grid">
  <section>
    <div class="container AND container--grid gap--responsive">
      <div class="container__full">container__full</div>
      <div class="container__center--md">container__center--md</div>
      <div class="container__center--sm">container__center--sm</div>
      <div class="container__center--xs">container__center--xs</div>
    </div>
  </section>
</Canvas>

## Main and Aside sub-containers

- `container__main` is left aligned per default, use it in combination with `container__aside` 
- `container__aside` is right aligned per default
- use `container--grid container--reverse` on the parent to swap `container__main` and `container__aside`

<Canvas className="storybook-demo-grid">
  <section>
    <div class="container AND container--grid gap--responsive">
      <div class="container__main">container__main</div>
      <div class="container__aside">container__aside</div>
    </div>
  </section>
</Canvas>

<Canvas className="storybook-demo-grid">
  <section>

    </div>

</Canvas>
